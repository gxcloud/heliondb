# HelionDB

HelionDB is a fast SQL database with PostgreSQL-compatible syntax, selectable per-table storage engines, async WAL persistence, and QUIC transport.

## Documentation

The full documentation now lives in the mdBook site and is published from this repository via GitHub Pages:

https://gxcloud.github.io/heliondb/

## Development

```bash
cargo build
cargo test
cargo clippy
```
